- Переменные, процедуры, функции: ВерхнийРегистрДляКаждогоСлова
- Константы: ВЕРХНИЙ_РЕГИСТР_С_ПОДЧЕРКИВАНИЯМИ
- Параметры процедур: РегистрКакУПеременных
