# Тестовый документ для проверки Docling

## Введение

Это тестовый документ для проверки работы MCP Docling сервера в рамках интеграции с 1C-Enterprise Framework.

## Основные функции

1. **Преобразование документов** - конвертация PDF, DOC, HTML в Markdown
2. **OCR поддержка** - распознавание отсканированных документов  
3. **Кэширование** - сохранение результатов для ускорения повторных запросов
4. **Batch обработка** - массовая обработка документов

## Архитектура

### MCP Server
- **Протокол:** Model Context Protocol
- **Транспорт:** stdio/SSE
- **Кэш:** ~/.cache/mcp-docling/

### Интеграция с 1C
- Memory MCP для сохранения в Knowledge Graph
- Task Master для управления задачами обработки
- BSL анализ результатов

## Заключение

Данная интеграция позволит автоматизировать обработку документов в рамках 1C-Enterprise разработки.
