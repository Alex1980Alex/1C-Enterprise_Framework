
Обработка гкс_АРМПромежуточныйКомпозит вкладка на форме СтраницаУсиленныйКонтроль

1. В АРМ по композитным пробам для УК выводим состояние регистрации (срез последних по РС_Состояния регистрации).

Заполнение данных для арма:
Подбираются из РС Показатели качества усиленного контроля с назначением Использования - промежуточный композит с учетом отборов указанных а Арм-е.
в РС гкс_СоставГруппТранспорта для регистраций ПЛК получаем информацию по группе ТС , куда она включена (невидима для пользователей).

Выводятся те регистрации, где состояние качества (новый реквизит) по усиленному = Пустое , отбор проб.
Если устанавливается галка "показывать все", то выводятся данные вне зависимости от состояния качества.

При интерактивной работе с таблицей анализов.
При выделении одной строки в ТЧ автоматически выделять все строки, где одинаковая Группа ТС и показатель. Снимать выделение по такому же признаку.

Если состояние качества усиленного контроля по регистрации не равно пустое значение или Отбор проб, то выводится сообщение пользователю, что анализы уже сделаны на показатели и повторно не могут быть сделаны. Форма далее не открывается (согласовать сообщение).

Показатель по выделенным строкам передаем на форму для формирование номера пробы и далее в лаб. анализ.

Если Качество не принято для регистрации, то не включаем в композитную пробу такую регистрацию. Но для такой регистрации в РС гкс_ПоказателиКачестваУсиленногоКонтроля пишем состояние "НеПринятоВходнойКонтроль".

2. Если в лаб. анализе тип Документа - регистрации Формирование номера проб.
То для показателей усиленного контроля поиск анализов / значений для определения качества происходит по регистрациям, которые входят в формирование номера пробы.
