# MCP Clipboard Server

## Описание

Пользовательский MCP сервер для работы с буфером обмена, созданный для фреймворка 1C-Enterprise_Framework.

## Возможности

- `clipboard_get_text` - получение текста из буфера обмена
- `clipboard_set_text` - запись текста в буфер обмена
- `clipboard_get_image` - получение изображения из буфера обмена и сохранение во временный файл

## Установка

Сервер уже настроен в конфигурации Claude Code:

```json
{
  "clipboard": {
    "type": "stdio",
    "command": "node",
    "args": ["D:/1C-Enterprise_Framework/mcp-clipboard-server/index.js"],
    "env": {
      "MCP_DEBUG": "false",
      "MCP_CLEANUP_TTL": "3600"
    }
  }
}
```

## Использование

После перезапуска Claude Code будут доступны инструменты:

- `mcp__clipboard__clipboard_get_text` - чтение текста из буфера
- `mcp__clipboard__clipboard_set_text` - запись текста в буфер
- `mcp__clipboard__clipboard_get_image` - извлечение изображения из буфера

## Интеграция с командой /clipboard-image

Этот MCP сервер работает совместно с пользовательской командой `/clipboard-image` для обеспечения полной функциональности работы с буфером обмена.
